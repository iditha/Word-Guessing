## Scoring System

The player's final score is calculated based on game performance using the following rules:

- Start with **100 points**.
- Subtract **1 point for every 5 seconds** that pass.
- Subtract **10 points** if a hint was used.
- Add **5 points** for each **remaining attempt**.
- The score is **never negative** (minimum is 0).
